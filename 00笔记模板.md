学习时生成学习总结笔记的要求（只针对学习总结笔记）： 
1. 顶部固定先写：这篇讲什么、先看这 3 条、我上次最容易错的点、30 秒自测。
2. 正文固定再写：核心规则、关键易错点、典型例题、判断流程、一句话总结、小练习、明天衔接。
3. 风格要简洁、可复习、重点前置，不要写成长篇流水账。
4. 解释要紧贴代码和我的错误点，不要只讲空泛概念。
5. 所有 Markdown 代码块必须成对闭合，绝不能漏掉结尾的三个反引号。
